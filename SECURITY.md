# Security Policy

Smart Presence V4 Premium keeps your data safe with offline-first architecture.

## How We Protect Your Data

**🔐 Authentication**
- JWT tokens for secure login
- Argon2 password hashing
- Admin and Staff roles only

**🛡️ Data Storage**
- Everything stored locally (SQLite)
- No cloud, no data sharing
- Biometric data never leaves your device

**🔒 API Security**
- Bearer token authentication
- Input validation
- Secure CORS setup

## Report Security Issues

Found a vulnerability? Email us: security@smartpresence.dev

We'll respond within 48 hours.

## Best Practices

- Use strong passwords
- Keep software updated
- Monitor staff accounts

## Compliance

Built to comply with GDPR, FERPA, and local privacy laws.
