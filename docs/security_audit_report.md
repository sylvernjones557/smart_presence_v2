## 4. Hardware Optimization (Low-Power)
The system is explicitly tuned for low-end hardware without GPU:
- **Image Pre-processing:** Automatic downscaling (640px max) to reduce CPU load.
- **Model Choice:** `buffalo_sc` (Small/Low-Memory) verified as the active model.
- **Thread Management:** Forced multi-threading limits (OMP_NUM_THREADS) to prevent system freeze.
- **Lazy Loading:** Face Engine only initializes upon first usage to save background RAM.
